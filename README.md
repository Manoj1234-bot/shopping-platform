# ShopSphere 🛒

ShopSphere is a dynamic e-commerce web application developed using PHP, MySQL, HTML, CSS, and JavaScript. The platform allows users to browse products, search items, and explore categories through a clean and interactive interface.

## 🌐 Live Website
https://shopsphere.page.gd

## 🚀 Features
- User authentication (Login & Registration)
- Product browsing and search functionality
- Category-based product filtering
- Dynamic product display using PHP and MySQL
- Responsive and user-friendly interface

## 🛠 Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## ⚙️ Development Environment
- XAMPP
- phpMyAdmin

## 📚 Purpose
This project was developed as part of my learning in web development during my Bachelor of Computer Applications (BCA). It demonstrates backend integration, database management, and dynamic web application development.
